# Study Language Academy Portal

**A full-stack multilingual language academy management system** built for education institutions in East Africa.

![Hero Banner](hero-banner.png)

## ✨ About the Project

**Study Language Academy Portal** is a complete web-based platform for managing a modern language school. It supports multiple user roles, online examinations, student management, finance tracking, and more.

- **Location**: Nairobi, Kenya
- **Developer**: Tom M. Mutua
- **Status**: Actively developed (2025–2026)

## 🚀 Key Features

- **Multi-role Dashboard** — Student, Staff, Chief Admin, Recruiter
- **Online Examination System** — Timed MCQs with automatic grading
- **Firebase Integration** — Real-time data sync (optional)
- **Fully Responsive** — Excellent mobile & tablet support
- **Multilingual Support** — English, Swahili, French, Arabic + 5 more
- **Offline Mode** — Works perfectly with localStorage
- **Recruiter Portal** — Referral and commission management

## 🛠 Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Firebase (Auth + Firestore)
- **Hosting**: Firebase Hosting / GitHub Pages
- **Database**: Firestore + localStorage fallback
